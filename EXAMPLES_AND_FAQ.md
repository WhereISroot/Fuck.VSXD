# Примеры использования и FAQ

## 🚀 Быстрые примеры

### Пример 1: Базовое использование через GUI

```bash
# Запустить приложение
python3 schematic_converter.py

# 1. Нажмите "📂 Загрузить изображение"
# 2. Выберите фотку вашей схемы (JPG или PNG)
# 3. Нажмите "⚙️ Конвертировать в VSDX"
# 4. Выберите папку для сохранения
# 5. Готово! Получите VSDX + JSON файлы
```

### Пример 2: Командная строка

```bash
# Обработать изображение и получить статистику
python3 advanced_recognition.py photo_of_scheme.jpg

# Результат:
# 📊 Статистика распознавания:
#   Всего элементов: 24
#   Проводов: 12
#   Компонентов: 8
#   Текста: 4
#   Узлов: 5
#   Среднее качество: 78.5%
```

### Пример 3: Конвертирование JSON в VSDX

```bash
# Если у вас есть JSON файл, конвертируйте его в VSDX
python3 vsdx_generator.py scheme.json scheme.vsdx
```

## ❓ Часто задаваемые вопросы

### В: Программа не запускается, пишет ошибку про PyQt5

**О:** Установите PyQt5:
```bash
sudo pacman -S python-pyqt5
# или
pip install PyQt5
```

### В: Tesseract не найден

**О:** Установите Tesseract OCR:
```bash
sudo pacman -S tesseract
```

Если даже после установки не работает, добавьте путь в код:
```python
pytesseract.pytesseract.pytesseract_cmd = r'/usr/bin/tesseract'
```

### В: Разпознавание работает плохо, много ошибок

**О:** Это нормально на первый раз! Попробуйте:
- ✅ Лучшее освещение при фото
- ✅ Фото в анфас, не под углом
- ✅ Контрастная схема (чёрное на белом лучше)
- ✅ Более высокое разрешение фото

Обычно 70-80% элементов распознаются правильно, остальное доделаете за 5 минут.

### В: Я хочу использовать это на Windows

**О:** 
1. Скачайте Python 3.9+ с python.org
2. Запустите `install_windows.bat`
3. Запустите `python schematic_converter.py`

### В: Можно ли обрабатывать несколько файлов сразу?

**О:** Пока нет, но можно сделать скрипт:

```bash
#!/bin/bash
for file in *.jpg; do
    python3 advanced_recognition.py "$file"
done
```

Или добавить массовую обработку в GUI (я это добавлю в версию 2.0).

### В: VSDX файл создаётся, но не открывается в Visio

**О:** Попробуйте:
1. Используйте JSON файл вместо VSDX
2. Откройте JSON в текстовом редакторе
3. Скопируйте нужные элементы в Visio вручную
4. Это займёт 5-10 минут вместо перерисовки на час

### В: Какие форматы поддерживаются?

**О:** На вход:
- JPG, JPEG
- PNG
- BMP

На выход:
- VSDX (Microsoft Visio)
- JSON (универсальный формат)

### В: Программа зависает при обработке большого файла

**О:** Это нормально для больших изображений (3000x4000px+). Подождите или:
- Уменьшите разрешение фото
- Используйте более мощный компьютер
- Обработка идёт в отдельном потоке, интерфейс не замерзает

### В: Можно ли интегрировать это в свой код?

**О:** Да! Используйте как библиотеку:

```python
from advanced_recognition import ElectricalSchematicRecognizer
from vsdx_generator import json_to_vsdx

# Распознавание
recognizer = ElectricalSchematicRecognizer()
elements = recognizer.process('my_scheme.jpg')

# Сохранение в JSON
import json
with open('result.json', 'w') as f:
    json.dump({'elements': elements}, f)

# Конвертирование в VSDX
json_to_vsdx('result.json', 'result.vsdx')
```

### В: Какая лицензия на код?

**О:** Используйте как хотите! MIT лицензия (или просто копируйте).

## 🔧 Продвинутые советы

### Улучшение качества распознавания

Если много ошибок, отредактируйте JSON перед конвертированием:

```json
{
  "elements": [
    {
      "type": "wire",
      "points": [[100, 100], [200, 200]],
      "confidence": 0.95
    },
    // Добавьте сюда исправления вручную
  ]
}
```

### Автоматизация с cron

Обрабатывайте схемы по расписанию:

```bash
# Добавьте в crontab -e
0 12 * * * cd /path/to/converter && python3 advanced_recognition.py *.jpg
```

### Интеграция с облаком

Используйте с облачным хранилищем:

```bash
# OneDrive, Google Drive или Dropbox
ln -s ~/Dropbox/schemes ~/converter_input
python3 run.sh
```

## 🐛 Решение проблем

### "ModuleNotFoundError: No module named 'cv2'"

```bash
# Переустановите OpenCV
pip install --upgrade opencv-python
```

### "Permission denied" при запуске .sh

```bash
# Сделайте файл исполняемым
chmod +x run.sh install_arch.sh
```

### Программа медленная

- Закройте другие приложения
- Используйте PNG вместо JPEG (быстрее парсится)
- Уменьшите разрешение фото

### Память закончилась при обработке

- Используйте меньшие файлы
- Обработайте схему по частям
- Перезагрузитесь
