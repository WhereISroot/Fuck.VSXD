## 🚀 КАК ИСПОЛЬЗОВАТЬ

### Вариант 1️⃣: Графический интерфейс (рекомендуется)
```bash
# Установка
pip install -r requirements.txt

# Запуск
python3 schematic_converter.py

# В окне программы:
# 1. Загрузить изображение
# 2. Выбрать форматы (CSV и/или VSDX)
# 3. Нажать "Конвертировать"
# 4. Выбрать место сохранения
```

### Вариант 2️⃣: Командная строка (CLI)
```bash
# Быстрое тестирование
python3 test_contours.py schema.jpg

# Это создаст:
# - schema_contours.csv
# - schema_parts.csv
# - schema_vertices.csv
# - schema_contours.json
# - schema_visualization.jpg
```

### Вариант 3️⃣: В своем коде (Python API)
```python
from contour_extractor import ContourExtractor

extractor = ContourExtractor('schema.jpg')
extractor.load_image()
extractor.preprocess_image()
contours = extractor.extract_contours()
parts = extractor.split_contours_into_parts()

# Экспорт
extractor.export_to_csv('output.csv')
extractor.export_to_json('output.json')
```

## 📁 ВЫХОДНЫЕ ФАЙЛЫ

Для изображения `schema.jpg` будут созданы:

### CSV файлы (для Microsoft Drawing)
| Файл | Содержание |
|------|-----------|
| `schema_contours.csv` | Основные параметры контуров (площадь, периметр, центр и т.д.) |
| `schema_parts.csv` | Отдельные части контуров с центроидами и границами |
| `schema_vertices.csv` | Все координаты вершин для воссоздания фигур |

### Другие форматы
| Файл | Содержание |
|------|-----------|
| `schema_contours.json` | Полные структурированные данные |
| `schema_visualization.jpg` | Изображение с нарисованными контурами |
| `schema.vsdx` | Файл для Microsoft Visio (опционально) |

## 🔍 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Пример 1: Базовая обработка
```python
from contour_extractor import ContourExtractor

extractor = ContourExtractor('circuit.jpg')
extractor.load_image()
extractor.preprocess_image()
contours = extractor.extract_contours(min_area=50)
extractor.export_to_csv('output.csv')
```

### Пример 2: Рисование из CSV
```bash
python3 csv_drawer.py circuit_vertices.csv result.jpg
```

### Пример 3: Использование в коде
```python
from csv_drawer import CSVShapeDrawer

drawer = CSVShapeDrawer(2000, 2000)
drawer.load_vertices_csv('vertices.csv')
drawer.draw_shapes(line_color=(0,0,255), line_thickness=2)
drawer.save('output.jpg')
```
