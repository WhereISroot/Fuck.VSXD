🚀 КАК НАЧАТЬ (GETTING STARTED):
═════════════════════════════════════════════════════════════════

ВАРИАНТ 1: Графический интерфейс (GUI) ⭐ РЕКОМЕНДУЕТСЯ
─────────────────────────────────────────────────────

1. Установите зависимости:
   $ pip install -r requirements.txt

2. Запустите программу:
   $ python3 schematic_converter.py

3. В окне программы:
   • Загрузите изображение
   • Выберите форматы (CSV и/или VSDX)
   • Нажмите "Конвертировать"


ВАРИАНТ 2: Командная строка (CLI)
──────────────────────────────────

1. Простое тестирование:
   $ python3 test_contours.py /path/to/schema.jpg

   Создаст все файлы в текущей директории

2. Для рисования из CSV:
   $ python3 csv_drawer.py schema_vertices.csv result.jpg


ВАРИАНТ 3: В своем коде (Python API)
─────────────────────────────────────

from contour_extractor import ContourExtractor

extractor = ContourExtractor('schema.jpg')
extractor.load_image()
extractor.preprocess_image()
contours = extractor.extract_contours()
parts = extractor.split_contours_into_parts()

extractor.export_to_csv('output.csv')
extractor.export_to_json('output.json')
extractor.visualize_contours('output.jpg')

❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ (FAQ):
═════════════════════════════════════════════════════════════════

Q: Какой формат использовать - CSV или VSDX?
A: CSV для Microsoft Drawing, VSDX для Visio

Q: Почему контуры не детектируются?
A: Проверьте контрастность изображения, попробуйте улучшить качество

Q: Как использовать CSV в Microsoft Drawing?
A: Используйте файл schema_vertices.csv и импортируйте координаты

Q: Могу ли я вызвать это из своего кода?
A: Да, используйте ContourExtractor класс из contour_extractor.py

Q: Сколько времени занимает обработка?
A: Обычно 5-15 секунд, в зависимости от размера изображения

Q: Где найти документацию?
A: README_UPDATED.md или quick_start_guide.py


📞 ТЕХНИЧЕСКИЕ ДЕТАЛИ:
═════════════════════════════════════════════════════════════════

Пороги обработки (в contour_extractor.py):
• min_area = 50 пиксели² (минимальная площадь контура)
• gap_threshold = 50 пиксели (минимальный разрыв для разделения)

Алгоритмы:
• cv2.adaptiveThreshold - адаптивная пороговая обработка
• cv2.morphologyEx - морфологические операции
• cv2.findContours - поиск контуров
• cv2.approxPolyDP - аппроксимация контуров
