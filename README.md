# Fuck VSDX

FuckVSDX - это проект созданный для автоматического преобразования фотографий схем в файлы VSDX (для Microsoft Visio)

**Суть:** Прикладываете фотку схемы → получаете VSDX файл, который можно сразу кинуть в Visio.
Сейчас реализованно для Arch и Windows.(автор ленивый студент, не просите ничего)
---

## Быстрый старт

### Для Arch Linux

```bash
# 1. Делаем скрипт исполняемым
chmod +x install_arch.sh

# 2. Запускаем установку
./install_arch.sh

# 3. Стартуем
python3 schematic_converter.py
```

### Для Windows (позже допилю)

```batch
# Просто запустить:
python schematic_converter.py
```

---

## Что делает программа

1. **Загружает фотографию** электрической схемы (JPG или PNG)
2. **Обнаруживает элементы:**
   - Провода
   - Компоненты 
   - Фунциональные рисунки(реле, ключ и пр.)
   - Подписи

free_hears3-2392711753_visualization.jpg

3. **Генерирует:**
   - VSDX файл (готовый для Visio)
   - JSON файл (для ручной доработки)

---

## Как использовать

1. Откройте программу
2. Нажмите кнопку "Загрузить изображение"
3. Выберите фотку вашей схемы
4. Нажмите "онвертировать в VSDX"
5. Выберите, где сохранить файл
6. Гойда

---

## Результаты

### Что вы получите:

- `scheme.vsdx` - готовый файл для Visio
- `scheme.json` - данные всех элементов (для ручной доработки)

### Точность распознавания

- Линии (провода): ~85-90%
- Фигуры: ~75-80%
- Текст: ~70-85% (зависит от качества фото)

**Обычно достаточно 10-20% доделок вручную вместо того чтобы две пары сидеть рисовать эту п@^&стику**

## Требования

- Python 3.7+
- OpenCV
- PyQt5(linux)
- Tesseract OCR
- PIL/Pillow
- NumPy

### На Arch всё устанавливается одной командой!

```bash
./install_arch.sh
```

---

## Рекомендации для лучших результатов

**Выбирайте** качественные фотографии:
- Много пикселей
- Изображение не под углом
- Чёткие линии
- Контрастная схема на белом фоне

 **НЕ ДАЙ б%@ боже**:
- Размытая фотка
- Темно как в.. 
- Фото под углом
- Схема с очень мелким текстом

## Что с этим можно сделать?

После конвертирования вы можете:

1. **Открыть в Visio** и подредактировать
2. **Открыть в LibreOffice Draw** и доделать
3. **Использовать JSON** для программной обработки
4. **Копировать элементы** в свои документы

## Решение проблем

### Программа не запускается
```bash
# Проверьте Python
python3 --version

# Переустановите зависимости
pip install --upgrade pip
pip install -r requirements.txt
```

### Tesseract не найден
```bash
# На Arch:
sudo pacman -S tesseract

```

### VSDX не создаётся, только JSON
JSON содержит все данные схемы. Вы можете:
- Доделать в Visio вручную
- Запустить дополнительный скрипт для преобразования JSON → VSDX
